# 05.01 Stream Viewing and Log Tailing

Execute basic audits of file payloads directly to the terminal stdout.

## Real World Scenario: Active Log Auditing
During a persistent denial-of-service attack on a web server, an administrator cannot afford to download huge log files. They must view malicious IP blocks in real time natively from the terminal.
1. Jump to the bottom of the active access log securely: `tail -n 20 /var/log/apache2/access.log`
2. Follow the stream live to witness attacking requests as they hit the server: `tail -f /var/log/apache2/access.log`
3. Exit the live stream using `Ctrl`+`C`.

## Unbounded Dumps
* `cat <filename>` : Dump complete file contents to stdout. Do not use this for extremely large logs, as it floods and crashes fragile terminal buffers.
* `cat <file1> <file2>` : Concatenate two distinct files sequentially. Ensure you understand that this merges pipelines purely to standard output and does not corrupt or alter the source files on disk.

## Paged Viewports
* `less <filename>` : Initialize an interactive paged buffer. You can scroll forward and backward. It explicitly avoids loading the entire file at startup, making it the only secure tool for querying gigabyte-scale logs.
* `more <filename>` : Legacy pager. Operates strictly page-by-page. Downward navigation only.

## Targeted Extraction
* `head -n <lines> <filename>` : Extract only the topmost `<lines>` data rows from the start of the file. Prefix `-n 10` extracts exactly the first 10 lines.
* `tail -n <lines> <filename>` : Extract the last `<lines>` of data. Useful for tracking end-of-file append updates.
* `tail -f <filename>` : Follow the file continuously, outputting new lines specifically as they are generated by external programs.

## Metric Processing
The `wc` (word count) utility quantifies payloads structurally natively.
* `wc <filename>` : Return strict counts of line, word, and byte structures within the file.
* `wc -l` : Output the active line count strictly. Excellent for quantifying records (e.g. `cat emails.txt | wc -l`).
