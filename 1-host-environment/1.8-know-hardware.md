# Know your hardware

In previous notes, you got to know how to find a command and it's detail. Let's now know about our machine on which the Linux is running.🤔

Here we will touch the host, kernel, CPU, memory, disks, sessions, and hardware surfaces.

## OS Information

```bash
# Information about the kernel
# (name, version, release, date etc.)
uname -a
# or
cat /proc/version

# Hostname OS MetaData
hostnamectl

# Distro's Information such as Code name, Release version etc.
lsb_release -a
```

## Networking related information

Suppose you have hosted a website from your system on localhost and want to impress someone or don't want to disclose your local IP address, then you can share your FQDN instead of your local IP address.

For example, instead of opening your website at `http://192.168.1.35/index.html`, user may use `http://host.example.com/index.html`. Doesn't it seem cool to you! 😉

```bash
# hostname, Full Qualified Domain name - FQDN
hostname
hostname -f

# Change the hostname
sudo hostnamectl set-hostname "new_hostname"

# Change FQDN and it's shorter version
sudo hostnamectl set-hostname --static "host.example.com"
sudo hostnamectl set-hostname --pretty "new_hostname"

# Reset hostname to its default value
sudo hostnamectl set-hostname --pretty ""

# local and public IP address
hostname -i
hostname -I

# Network card information
lshw -c network
```

## Hardware information

```bash
# Brief CPU information
sudo lshw -c processor

# CPU information in doc format
lscpu

# CPU information in JSON format
lscpu --json

# List of all CPUs. Easy way to get number of CPU in the System.
# If you have a multi-core processor, you can see how many you actually have.
lscpu --extended

# CPU Cache memory
lscpu --caches

# Detailed CPU information
# (including each processor detail separately)
cat /proc/cpuinfo
```

## Memory and Disk information

```bash
# Memory (RAM) and SWAP state (Total, Used, Free, etc.)
free -h

# Filesystem (Disks) Capacity (Size, Used, Available, etc.)
df -h

# List information about Disks
lsblk
```

## More information

```bash
# List a particular class of hardware:
sudo lshw -class system|bridge|memory|processor|address|storage|disk|tape|bus|network|display|input|printer|multimedia|communication|power|volume|generic

# General information about the System
sudo lshw -c generic
```


## Practical usecases

*Extract CPU model for a report:*

```bash
lscpu | grep -i 'model name' | cut -d: -f2 | sed 's/^ *//'
# 13th Gen Intel(R) Core(TM) i7-13620H
```
