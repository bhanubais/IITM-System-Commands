# 04.02 nano Operations and Log Reading

The `nano` editor engineers a continuously visible interface. Operations require direct chording with `Ctrl` (Command) or `Alt` (Meta) keys. It is optimal for rapid, simplified edits.

## Real World Scenario: Injecting API Keys
During a hackathon, you must deploy an `.env` file containing payment gateway API keys without exposing them to visual onlookers or saving them in shell history.

1. Create and open the file securely: `nano .env`
2. Type the sensitive variables directly into the buffer.
3. Save the payload: `Ctrl`+`O`, followed by `Enter`.
4. Escape the interface securely: `Ctrl`+`X`.

## File Pipeline
* `Ctrl`+`O` : Write buffer to file (Save As equivalent).
* `Ctrl`+`S` : Commit save to the active file.
* `Ctrl`+`R` : Inject external file data directly into the active buffer. This allows appending data from multiple files directly.
* `Ctrl`+`X` : Terminate session and exit.

## Block Operations and Edits
* `Ctrl`+`K` : Cut the active line into the internal cutbuffer. Used heavily to delete extraneous config lines.
* `Alt`+`6` : Copy the active line into the cutbuffer.
* `Ctrl`+`U` : Paste the contents of the cutbuffer exactly at the cursor.
* `Ctrl`+`H` : Delete character backward.
* `Ctrl`+`D` : Delete character at cursor.
* `Alt`+`Del` : Delete active line immediately.
* `Alt`+`3` : Toggle line/region comments.
* `Alt`+`U` / `Alt`+`E` : Execute undo and redo queues.

## Navigation and Search Matrices
* `Ctrl`+`W` : Base search operation (forward lookup).
* `Ctrl`+`Q` : Execute backward search operation.
* `Alt`+`W` / `Alt`+`Q` : Cycle to next search occurrence forward or backward.
* `Alt`+`R` : Initialize global search and replace protocol.
* `Ctrl`+`Y` / `Ctrl`+`V` : Scroll viewport page down or page up.
* `Ctrl`+`C` : Audit immediate cursor coordinate position. Highly beneficial when hunting parser errors at specific line coordinates.
