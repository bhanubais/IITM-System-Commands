# 08.01 Network Diagnostics

Evaluate network pathways testing the operational boundaries mapping local and external domain connections.

## Connection Parameters
* `ping <domain>` : Inject ICMP sequence packets confirming target routing infrastructure remains active. Terminate execution with `CTRL+C`.
* `traceroute <domain>` : Trace specific network topography bounds tracking every node hop jumping to an endpoint.
* `netstat -tulpn` : Diagnose all listening sockets alongside active transport layer pipelines (TCP/UDP) mapping PID architecture.

## Address Resolutions
* `ifconfig` or `ip addr show` : Output network hardware interfaces evaluating bound internal IP targets.
* `whois <domain>` : Execute DNS evaluations tracking the absolute host registry records globally.
* `nslookup <domain>` : Extract basic DNS array configurations mapping URLs to binary IP numbers.
