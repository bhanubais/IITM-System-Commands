# 04.02 nano Operations and Log Reading

`nano` is a simple terminal editor with always-visible shortcuts. It is a good choice when you want quick edits without leaving the shell.

## Real World Scenario: Injecting API Keys
During a hackathon, you must deploy an `.env` file containing payment gateway API keys without exposing them to visual onlookers or saving them in shell history.

1. Create and open the file securely: `nano .env`
2. Type the sensitive variables directly into the buffer.
3. Save the payload: `Ctrl`+`O`, followed by `Enter`.
4. Escape the interface securely: `Ctrl`+`X`.

## File Pipeline
* `Ctrl`+`O` : Write the buffer to disk.
* `Ctrl`+`S` : Save in place.
* `Ctrl`+`R` : Read another file into the current buffer.
* `Ctrl`+`X` : Exit `nano`.

## Block Operations and Edits
* `Ctrl`+`K` : Cut the active line into the cutbuffer.
* `Alt`+`6` : Copy the active line into the cutbuffer.
* `Ctrl`+`U` : Paste the cutbuffer at the cursor.
* `Ctrl`+`H` : Delete the character before the cursor.
* `Ctrl`+`D` : Delete the character under the cursor.
* `Alt`+`Del` : Delete the current line.
* `Alt`+`3` : Toggle comments on the selected text.
* `Alt`+`U` / `Alt`+`E` : Undo and redo.

## Navigation and Search Matrices
* `Ctrl`+`W` : Search forward.
* `Ctrl`+`Q` : Search backward.
* `Alt`+`W` / `Alt`+`Q` : Move between matches.
* `Alt`+`R` : Search and replace.
* `Ctrl`+`Y` / `Ctrl`+`V` : Scroll viewport page down or page up.
* `Ctrl`+`C` : Show the current cursor position.

## Quick Reference (Shortcuts)
Common nano shortcuts compressed for quick scanning during edits:

File handling
- `Ctrl+S` : Save
- `Ctrl+O` : Write file (Save As)
- `Ctrl+X` : Exit

Editing
- `Ctrl+K` : Cut line
- `Alt+6` : Copy line
- `Ctrl+U` : Paste
- `Alt+3` : Toggle comment on region

Navigation
- `Ctrl+A` / `Ctrl+E` : Line start / end
- `Ctrl+Y` / `Ctrl+V` : Page up / down
- `Alt+G` : Go to line

Notes:
- `Ctrl+C` reports cursor position; useful when correlating with compiler or log line numbers.
