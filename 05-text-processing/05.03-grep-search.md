# 05.03 Text Search with Grep

## Purpose

Search files or streams and print matching lines.

Use `grep` for direct log, code, and report inspection.

## Strict Syntax

```bash
grep 'pattern' <file>
grep [options] 'pattern' <file>
grep -E 'extended_pattern' <file>
egrep 'extended_pattern' <file>
```

## Command Map

| Command | Use |
|---|---|
| `grep` | Search with Basic Regular Expressions by default. |
| `grep -i` | Ignore case. |
| `grep -v` | Invert match. |
| `grep -c` | Count matching lines. |
| `grep -l` | Print names of files with matches. |
| `grep -E` | Use Extended Regular Expressions. |
| `egrep` | Legacy spelling for `grep -E`. |

## Atomic Examples

```bash
grep 'FAILED root' /var/log/auth.log
grep -i 'failed' /var/log/auth.log    # ignore case
grep -v '^$' script.sh                # remove empty lines
grep -c 'pattern' file.txt            # count matching lines
grep -l 'pattern' *.log               # list files containing matches
grep '^e' names.txt                   # lines starting with e
grep 'am$' names.txt                  # lines ending with am
grep -E '(ED|ME)' names.txt           # extended alternation
egrep '\b[[:alnum:]]+\.[[:alnum:]]+\b' input.txt # word-like dotted tokens
```

## Compound Scenarios

Filter failed logins while excluding a known workstation:

```bash
grep -i 'failed' /var/log/auth.log | grep -v '192.168.1.50'
```

Count C++ files containing a target signature:

```bash
egrep -l '^void[[:space:]]+main\(\)[[:space:]]+\{$' *.cpp | tee /dev/tty | wc -l
```

Find package names of exactly four characters:

```bash
dpkg-query -W -f='${Section} ${binary:Package}\n' | egrep ' .{4}$'
```

## Exam Traps

`grep '[error]'` does not search for the word `error`; it matches one character from the set.

`grep -v` prints nonmatching lines.

`grep -c` counts matching lines, not total matches.

`egrep` is equivalent to `grep -E` on modern systems.

## Production Notes

Pipe `grep` stages when each filter has a distinct operational purpose.

Use `grep -l` to find affected files before editing them.

Quote patterns.
