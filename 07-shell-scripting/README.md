* Predict expansion order and quoting behavior.
* Validate conditional expressions and exit status.
* Build loops and functions with positional parameters.

## Production Targets

* Write scripts that fail visibly.
* Preserve whitespace and filenames safely.
* Route output, error, and config payloads predictably.
