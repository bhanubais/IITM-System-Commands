# 01.04: Manuals and System Help

System governance requires precise documentation retrieval. Use these utilities to audit command structures and isolate operating parameters.

## Manual Pages
Execute the `man` command to parse standard manual pages. Manuals are strictly divided into specific sections.

* `man <command>` : Access the primary manual page for the instruction.
* `man <section> <command>` : Target a specific manual section (e.g., `man 1ssl passwd`).
* `man -f <command>` : Display a concise description alongside available sections.
* `man -k <keyword>` : Search manual page descriptions for a specific keyword. Same as `apropos`.

### Manual Sections
1. Executable programs or shell commands.
2. System calls provided by the Kernel.
3. Library calls.
4. Special files found in `/dev`.
5. File formats and conventions.
6. Games.
7. Miscellaneous (macro packages).
8. System administration commands.
9. Kernel routines.

## Alternative Documentation
* `info <command>` : Read extended hypertext documentation. Use `Spacebar` to page forward and `Tab` to navigate hyperlinks.
* `tldr <command>` : Retrieve community-driven practical examples for quick operational reference.
* `help` : Audit information exclusively for bash built-in commands.
* `/usr/share/doc/` : Directory containing supporting documents like copyright licenses and `README` files for installed packages.

## Command Localization
* `type <command>` : Identify the command classification (built-in, alias, or filesystem binary).
* `whereis <command>` : Isolate the binary, source, and manual page locations.
* `which <command>` : Locate the execution binary within the current user `$PATH`.
* `whatis <command>` : Extract a one-line functional description from the manual interface.
* `apropos <keyword>` : Execute a regex search against manual page names and descriptions.

## Quick Recipes
- Compare outputs of `env` and `printenv`, show unique lines:
```bash
echo -e "$(env)\n$(printenv)" | sort | uniq -u
```
- Compare `man -k` and `apropos`, show common and unique entries:
```bash
echo -e "$(man -k ^ls)\n$(apropos ^ls)" | sort | uniq -u   # unique
echo -e "$(man -k ^ls)\n$(apropos ^ls)" | sort | uniq -d   # common
```
