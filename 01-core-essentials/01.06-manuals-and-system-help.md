# 01.04 Manuals and System Help

## Purpose

Find exact syntax and command classification from the system itself.

Use manual pages, built-in help, and binary location tools before guessing flags.

## Strict Syntax

```bash
man <command>
man <section> <command>
man -f <command>
man -k <keyword>
apropos <keyword>
type <command>
which <command>
whereis <command>
```

## Command Map

| Command | Use |
|---|---|
| `man` | Read manual pages. |
| `man -f` | Show manual entries for a command. |
| `man -k` | Search manual descriptions. |
| `apropos` | Search manual names and descriptions. |
| `info` | Read hypertext documentation. |
| `tldr` | Read practical community examples. |
| `help` | Inspect Bash built-ins. |
| `type` | Classify command resolution. |
| `which` | Locate executable in `PATH`. |
| `whereis` | Locate binary, source, and manual files. |
| `whatis` | One-line manual description. Same as `man -f` |

## Atomic Examples

```bash
man ls
man 5 passwd
man -f passwd
man -k regex
apropos regex
apropos regex --long
info coreutils
tldr tar
help cd
type cd
type grep
which grep
whereis grep
whatis grep
```

## Manual Sections

| Section | Content |
|---|---|
| `1` | User commands. |
| `2` | System calls. |
| `3` | Library calls. |
| `4` | Special files. |
| `5` | File formats. |
| `6` | Games. |
| `7` | Miscellaneous. |
| `8` | Administration commands. |
| `9` | Kernel routines. |

## Compound Scenarios

Identify whether a command is built-in or external:

```bash
type cd
help cd
man bash
```

Compare manual search outputs:

```bash
man -k '^ls'
apropos '^ls'
```

Find where a command comes from:

```bash
type grep
which grep
whereis grep
```

## Exam Traps

Manual section numbers matter. `passwd(1)` and `passwd(5)` are different topics.

`help` documents Bash built-ins, not all system commands.

`which` does not explain aliases and shell functions as well as `type`.

## Production Notes

Use `man` for precise flags.

Use `type` before debugging command behavior hidden by aliases or functions.

Use `apropos` when the command name is unknown but the task is known.
