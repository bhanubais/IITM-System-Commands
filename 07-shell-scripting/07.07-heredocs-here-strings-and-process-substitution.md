# 07.07 Heredocs, Here-Strings, and Process Substitution

## Purpose

Feed bulk input to commands.

Turn command output into file-like streams.

Use these constructs when normal pipes cannot express the data shape cleanly.

## Strict Syntax

```bash
command <<EOF
text
EOF

command <<< "$value"
command < <(producer)
diff <(cmd1) <(cmd2)
command > >(consumer)
```

## Command Map

| Construct | Meaning | Portability |
|---|---|---|
| `<<EOF` | Heredoc. Multi-line stdin block. | POSIX shell supports heredocs. |
| `<<<` | Here-string. Single string to stdin. | Bash, Zsh, Ksh. |
| `<(command)` | Read process substitution. | Bash, Zsh, Ksh. |
| `>(command)` | Write process substitution. | Bash, Zsh, Ksh. |
| `mkfifo` | Named pipe fallback. | POSIX systems commonly provide it. |

## Atomic Examples

```bash
cat <<EOF
server=api01
port=443
EOF

grep -F 'error' <<< "$line"         # here-string feeds one string to stdin
wc -w <<< "$text"                   # here-string appends a newline
diff <(sort old.txt) <(sort new.txt) # compare generated streams as files
paste <(cut -f1 users.tsv) <(cut -f2 users.tsv) # join two generated streams
some_command 2> >(sed 's/^/ERR: /' >&2) # process stderr through sed
producer | tee >(gzip -c > data.gz) >(sha256sum > data.sha256) >/dev/null # split one stream to processors
```

## Compound Scenarios

Generate a config file from fixed content and variables:

```bash
host="api01"
port="443"
cat > app.conf <<EOF
HOST=$host
PORT=$port
EOF
```

Compare two transformed streams without temporary files:

```bash
diff <(grep -v '^#' old.conf | sort) <(grep -v '^#' new.conf | sort)
```

Use a named pipe when process substitution is unavailable:

```bash
tmpdir="$(mktemp -d)"
fifo="$tmpdir/input"
mkfifo "$fifo"
sort data.txt > "$fifo" &            # writer must run in background
uniq -c < "$fifo"                    # reader consumes FIFO
rm -r -- "$tmpdir"
```

## Exam Traps

Here-strings append a newline.

Quoted heredoc delimiters suppress expansion inside the block.

Process substitution is not POSIX.

Named pipes can block when a reader or writer is missing.

## Production Notes

Use heredocs for generated config and test payloads.

Use process substitution to compare transformed files without writing temporary artifacts.

Use `mktemp -d` when building FIFO fallbacks.
