# 01.05: System Time and Scheduling

Configure local system temporal settings and execute strict power cycle scheduling. Calculate chronometric data and query standard output.

## Chronometric Data
* `date` : Output the active system time and date parameters.
* `date -R` : Extract the time adhering strictly to RFC 5322 standard.
* `cal` : Dump the standard calendar matrix for the active month. Follow with parameters `[month] [year]` to target specific iterations (e.g., `cal apr 2026`).
* `ncal` : Output calendar arrays utilizing vertical orientation geometries.

## Lifecycle and Power State Governance
Execute scheduled power protocols and override active boot cycles.

* `shutdown -h now` : Abort all kernels and halt power immediately.
* `shutdown --reboot now` : Terminate kernel array and force structural reboot immediately.
* `shutdown --reboot +5` : Queue system reboot parameters with a strict 5-minute countdown.
* `shutdown -h 13:00` : Post hard halt protocol triggering precisely at 13:00 local metric time.
* `shutdown -c` : Retract and cancel any active pending shutdown logic branches.
