# Tailoring Shell Behavior

Several files in your home directory control the behavior of bash shells. These files have bash commands (known as bash scripts) those run upfront any of your action. For example:

- Run at logged in: `~/.bash_profile`, `~/.bash_login`, `~/.profile` etc.
- Upon launch of an interactive shell: `~/.bashrc`
- Run at logged out: `~/.bash_logout`

All of these files can set variables, run programs, print messages, or whatever you like.

We will use `~/.bashrc` frequently as we progress in the Linux journey.
