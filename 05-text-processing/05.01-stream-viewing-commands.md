# 05.01 Stream Viewing Commands

Execute basic audits of file payloads directly to the terminal stdout.

## Unbounded Dumps
* `cat <filename>` : Dump complete file contents to stdout. Do not use this for extremely large logs.
* `cat <file1> <file2>` : Concatenate two distinct files sequentially.

## Paged Viewports
* `less <filename>` : Initialize an interactive paged buffer. You can scroll forward and backward. Does not load the entire file at startup.
* `more <filename>` : Legacy pager. Operates strictly page-by-page.

## Targeted Extraction
* `head -n <lines> <filename>` : Extract only the topmost `<lines>` data rows from the start of the file. Prefix `-n 10` extracts the first 10 lines.
* `tail -n <lines> <filename>` : Extract the last `<lines>` of data. Useful for tracking end-of-file logs.

## Metric Processing
* `wc <filename>` : Return strict counts of line, word, and byte structures within the file.
* `wc -l` : Output the line count strictly.
