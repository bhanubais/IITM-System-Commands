# Getting Help

If you don't know where you should start, you may use `help` itself as a beginner. It will give you a list of available built-in commands, function names, and keywords you can start to look at.

## Get a starting point

```bash
❯ help
❯ info
```

## Search for a command

```bash
❯ man -k <keywords>
# or
❯ apropos <keywords>
```

## Brief about a command

```bash
# Single Line description for a command
❯ man -f <command> #or
❯ whatis <command>

# Classify a command whether it is build-in or a file-based command
❯ type <command>

# Getting location of a command
# It will provide you all the locations this command belongs to
❯ type -a <command>

# Getting location that is used by the current Shell
❯ which <command>

# Get the location where the manual about the command is stored
❯ whereis <command>
```

## Detail about a command

```bash
# Community driven short and practical usecase about a command
❯ tldr <command>

# detailed document
❯ man <command>

# Standard info documentation
❯ info <command>
```

If you want to know more about a **shell builtin** keywords, functions, or commands, you should use:

Here, a keyword may be a function name, reserved keyword, or built-in command. With the `help` command, you could find the information that you couldn't get from anywhere else. `help` command will be very useful while you will start learning shell scripting.

```bash
help <keyword>

# Example
❯ help for
❯ help if
❯ help wait
...
```


### Manual Sections

A manual may be divide in multiple parts. Following are few standard sections a command may have in it's man documentation.

| Section | Content |
|---|---|
| `1` | User commands. |
| `2` | Shell commands. |
| `3` | Library calls. |
| `4` | Special files (usually from `/dev`). |
| `5` | File formats and conventions. |
| `6` | Games. |
| `7` | Miscellaneous. |
| `8` | Administration commands (usually for root). |
| `9` | Kernel routines. |

```bash
❯ man <section-num> <command>

# Example:
❯ man 1 passwd
❯ man 5 passwd
# both will show different section for the command `passwd`
```
