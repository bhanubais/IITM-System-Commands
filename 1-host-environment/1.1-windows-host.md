# Windows Host Preparation

Before even enter the house of Linux, first get a similar experience in our house (Windows OS). We will install industry recognized Terminal tool for Windows eco-system.

## Install Powershell `pwsh`

Run the following command as per your requirement. If you already have `pwsh` just update it, if you don't have then install it. It's very easy.

```bash
# Get winget version
winget --version

# Check whether powershell already installed.
# If you already have it, just update it.
pwsh --version

# Search pwsh package correct ID
winget search PowerShell

# Show PowerShell Metadata
winget show Microsoft.PowerShell

# install PowerShell
winget install Microsoft.PowerShell

# Update winget itself
winget upgrade Microsoft.AppInstaller

```

## Basic `winget` commands

| Command | Use |
|---|---|
| `winget list` | List all installed packages. |
| `winget list --source winget` | List installed packages manageable through `winget`. |
| `winget search <package>` | Search for a new package. |
| `winget show <package_id>` | Inspect package metadata before install. |
| `winget install <package_id>` | Install a package. |
| `winget update` | List upgradable packages. |
| `winget upgrade <package_id>` | Upgrade one package. |
| `winget uninstall <package_id>` | Remove a package. |
