# 07.03 Control Structures

## Purpose

Branch and repeat shell logic.

Use control structures to turn command sequences into maintainable scripts.

## Strict Syntax

```bash
if condition; then ... elif condition; then ... else ... fi
case "$value" in pattern) ... ;; *) ... ;; esac
for item in list; do ... done
while condition; do ... done
select name in list; do ... done
```

## Command Map

| Structure | Use |
|---|---|
| `if` | Conditional branch. |
| `elif` | Additional branch. |
| `else` | Fallback branch. |
| `case` | Pattern-based branch. |
| `for` | Iterate fixed list or glob result. |
| `while` | Repeat while condition succeeds. |
| `select` | Build simple numbered menu. |
| `break` | Exit loop. |
| `continue` | Skip to next loop iteration. |

## Atomic Examples

```bash
if [[ $var -gt 3 ]]; then printf 'ok\n'; fi # Bash conditional expression
if [[ $var -eq 3 ]]; then printf 'equal\n'; else printf 'other\n'; fi # explicit fallback
case "$var" in start) printf 'start\n' ;; stop) printf 'stop\n' ;; *) printf 'unknown\n' ;; esac # pattern dispatch
for file in *.txt; do printf '%s\n' "$file"; done # iterates expanded glob results
for i in "${arr[@]}"; do printf '%s\n' "$i"; done # preserves each array element
while [[ $count -lt 5 ]]; do ((count++)); done # arithmetic increment loop
select x in alpha beta gamma; do printf '%s\n' "$x"; break; done # menu loop using PS3
```

## Compound Scenarios

Append all text files into one log:

```bash
for file in *.txt; do
	printf 'processing=%s\n' "$file"
	cat "$file" >> unified_logs.txt
done
```

Read CSV-like data line by line:

```bash
while IFS=, read -r field1 field2 rest; do
	printf '%s %s\n' "$field1" "$field2"
done < data.csv
```

Route subcommands through `case`:

```bash
case "$1" in
	start) ./service-start ;;
	stop) ./service-stop ;;
	*) printf 'usage: %s start|stop\n' "$0" >&2; exit 2 ;;
esac
```

## Exam Traps

`if` closes with `fi`.

`case` closes with `esac`.

Quote variables in `case` and loops when values may contain whitespace.

`select` uses `PS3` as its prompt.

## Production Notes

Use `case` for command dispatch.

Use `while IFS= read -r` for line-safe file input.

Avoid `for item in $(command)` when output can contain spaces.
