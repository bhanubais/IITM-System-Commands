# 03.01 Debian Software Governance

Execute structural control over external binaries using Advanced Package Tool (`apt`) and Debian Package management (`dpkg`). Software installation mutates system capability and requires absolute precision.

## Advanced Package Tool (APT)
APT functions as the primary high-level package resolution framework. Deploy systems sequentially.

### Server Deployment Example Pipeline
1. `apt-cache search apache2` : Query the package database targeting the server binary.
2. `sudo apt-get update` : Synchronize the local repository index before pulling code.
3. `sudo apt-get install apache2` : Deploy the target package directly onto the operating system.

### Core Commands
* `sudo apt-get upgrade` : Scale all active installed packages to their most recent secure versions.
* `apt-cache show -a <package_name>` : Extract granular package specification records.
* `sudo apt-get remove <package_name>` : Purge the package binary but preserve independent configuration files.
* `sudo apt-get purge <package_name>` : Exterminate both the target package and its internal configurations.
* `sudo apt autoremove` : Audit and execute removal of all decoupled, obsolete dependency packages.

## Base Debian Package (dpkg)
For offline installation and granular local auditing, leverage `dpkg`. Do not utilize `dpkg` for network resolution, as it bypasses external dependency graphs.

* `dpkg -l <pattern>` : List all locally installed packages conforming to the text pattern.
* `dpkg -L <package_name>` : Extract all precise filesystem paths generated by the target package.
* `dpkg -S <pattern>` : Reverse audit. Identify the parent package responsible for generating a specific installed file or path.
* `dpkg-query -W -f='${Section} ${binary:Package}\n'` : Format the query to extract highly specific data fields.
* `sudo dpkg -i <filename.deb>` : Directly deploy a local raw Debian package binary. External missing dependencies must be handled manually.

## Log Auditing
* `sudo tail -n 100 /var/log/auth.log` : Inspect authentication failures and authorization anomalies during package manipulation operations.
