# 03.99 LPG Sandbox Deployment

The Linux Pocket Guide (LPG) provides unique testing structures strictly meant for sandboxed environments. Do not integrate these payloads into live server hardware.

## Deployment Payload

This sequence grabs the compressed practice environment structure natively from the remote repository, unpacks it structurally directly into the local session path, and triggers the structural initiation executable logic.

```bash
curl -o https://linuxpocketguide.com/LPG4.tar.gz
tar -xf LPG4.tar.gz
bash reset-lpg
```

Execute this module specifically inside an isolated local folder tree.

Safety and reset notes:
- `bash reset-lpg` is provided by the LPG package to reset the sandbox to a known starting state. Run this only inside the extracted `linuxpocketguide` folder and never on production hosts.
- To re-download the practice files and reset the sandbox:
```bash
cd ~/linuxpocketguide
git checkout -- .   # discard local changes (if the sandbox is a git repo)
bash reset-lpg
```
